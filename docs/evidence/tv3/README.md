This directory contains evidence files collected during TV3 testing.

Evidence files are generated by running the attack and observability scripts.
Do not commit sensitive data (tokens, secrets) in evidence files.

Run to generate evidence:
  bash tests/attack/ssrf-attack.sh
  bash tests/attack/token-replay.sh
  bash tests/attack/webhook-forgery.sh
  bash tests/attack/bola-object-access.sh
  bash tests/attack/rate-limit-trigger.sh
  bash tests/metrics/measure-mttd-mttr.sh
  bash ci/run-local-security-scan.sh
  bash tests/zap/run-zap-baseline.sh
