This directory contains evidence files collected during TV2 testing.

Evidence files are generated by running the test and demo scripts.
Do not commit sensitive data (tokens, secrets) in evidence files.

Run to generate evidence:
  bash tests/auth/test-user-profile.sh
  bash tests/auth/test-order-access.sh
  bash tests/attack/bola-object-access.sh
  bash vault/scripts/init-dev-vault.sh
